# Image Naming Convention Guide

## SEO-Friendly Image File Naming Rules

Before uploading images to the website or Google Business Profile, rename files using descriptive, keyword-rich names based on the image content.

### Naming Pattern
```
[main-keyword]-[secondary-keyword]-[location]-[optional-descriptor].jpg
```

### Rules:
1. **Use lowercase letters only**
2. **Separate words with hyphens (-)**
3. **Include location** (karnal, gurgaon, delhi, etc.) when relevant
4. **Include main keywords** from the alt text
5. **Keep it descriptive but concise** (3-6 words typically)
6. **Remove articles** (a, an, the)
7. **Remove special characters** except hyphens

### Examples:

**Bad:** `IMG_1234.jpg`, `DSC_5678.jpg`, `photo1.jpg`

**Good:**
- `modern-false-ceiling-design-karnal.jpg`
- `gypsum-living-room-ceiling-led-lighting-karnal.jpg`
- `minimal-bedroom-ceiling-peripheral-led-karnal.jpg`
- `professional-office-grid-ceiling-karnal.jpg`
- `coffered-ceiling-design-living-room-karnal.jpg`
- `kitchen-moisture-resistant-ceiling-karnal.jpg`
- `luxury-master-bedroom-multi-layer-ceiling-karnal.jpg`

### Conversion from Alt Text to Filename

**Alt Text:** "Modern Gypsum False Ceiling Design for Living Room in Karnal with LED Lighting"
**Filename:** `modern-gypsum-false-ceiling-living-room-led-lighting-karnal.jpg`

**Alt Text:** "Minimal Gypsum False Ceiling Design for Bedroom in Karnal with Peripheral LED"
**Filename:** `minimal-gypsum-bedroom-ceiling-peripheral-led-karnal.jpg`

**Alt Text:** "Professional Grid False Ceiling Design for Office in Karnal with Integrated LED Panels"
**Filename:** `professional-grid-office-ceiling-integrated-led-panels-karnal.jpg`

### For Google Business Profile (GBP)

When uploading photos to Google Business Profile:
1. Rename the file BEFORE uploading
2. Use the same naming convention
3. Google's AI reads file names to understand image content
4. This helps with local SEO and image search rankings

### File Format
- Use `.jpg` for photographs
- Use `.png` for images with transparency
- Use `.webp` for optimized web images (recommended for website)

### Image Optimization
- Compress images before uploading (aim for <200KB per image)
- Use appropriate dimensions (typically 1200-2000px width for web)
- Maintain aspect ratio
- Use descriptive alt text that matches the filename
